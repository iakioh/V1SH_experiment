# V1SH_experiment
Psychophysics experiment on a neural circuit model of visual primary cortex (V1) for the V1 Saliency Hypothesis. Implemented during a lab rotation at Prof. Li Zhaoping's lab.

1. To replicate the experiment, download the standalone PsychoPy (version 2025.1.1) from https://docs.psychopy.org/. Use the builder view and open the psyexp files in the folder "experiement/code/". 

2. To replicate the data analysis, recreate the python environment with
``` pip install -r requirements.txt ```
and run the python notebooks in the folder "analysis/".
